PROGRAMMAUSFÜHRUNG
------------
Um das Spiel (Server und Client) lokal auf einem Rechner auszuführen, reicht es aus, zunächst die Datei ServerModul.jar und dann ClientModul.jar auszuführen. Im Fenster im Client kann dann 'lokales Spiel' gewählt werden.

Soll das Spiel auf mehreren Rechnern gespielt werden, muss zunächst der Server mit Konsole gestartet werden. Unter Windows geschieht dies mittels 'java -jar Servermodul.jar'. Dann lässt sich die IP Adresse des Servers aus der Konsole auslesen. Diese muss in das Fenster, welches sich beim Start der Datei ClientModul.jar öffnet, eingetragen werden und das Spiel kann gestartet werden.


Hinweis: Die Dateien benutzerDaten.xml und ServerModul.jar müssen sich im gleichen Ordner befinden.

Hinweis zum Registrieren:
1. Kein Spielername darf doppelt registriert werden.
2. Das Passwort muss aus 6 bis 24 Groß-, Kleinbuchstaben und Zahlen bestehen.
3. Der Name muss aus 3 bis 20 Zeichen bestehen.
